Update Percentiles and Averages

The server needs the charts uploaded as plain text. That means that paragraph marks and other formatting must be removed. The best way to do this is to copy/paste the PDF into MSWord. Only paste relevant data and none of the headers, etc. Next, use find/replace to search for the special character "^p" and replace that with a single space. Click replace all. You can paste the result into the textfield below. See example below for how this should look. Also, enter the earliest year listed on the form.
